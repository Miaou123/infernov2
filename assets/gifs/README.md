# GIF Assets for Twitter Bot

Place your animated GIFs here for the Twitter bot:

- `milestone.gif` - Attached to milestone burn tweets
- `buyback.gif` - Attached to buyback burn tweets

## Requirements
- Format: GIF
- Max size: 15MB (recommended under 5MB)
- Recommended dimensions: 480x480 or 600x400

## Ideas
- **milestone.gif**: Epic fire explosion, token burning animation
- **buyback.gif**: Money converting to fire, loop animation
